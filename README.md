# aliyun-specs
Aliyun Mobile Service CocoaPods specs.

阿里云移动服务iOS CocoaPods仓库。

## 使用
- Podfile中指定仓库位置，

```
source 'https://github.com/aliyun/aliyun-specs.git'
```

- 为工程target添加依赖，例：引入阿里云推送依赖，

```
pod 'AlicloudPush', '~> 1.9.1'
```
